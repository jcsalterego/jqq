# jqq

jq Querier: an interactive wrapper around `jq`

## Requirements

* [jq](https://github.com/stedolan/jq)
* Modern-ish ruby

## Installation

```
alias jqq="/path/to/jqq.rb"
```

## Usage

```
jqq . test.json
```
